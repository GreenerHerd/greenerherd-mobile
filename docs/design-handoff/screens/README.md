# Screen exports (Claude Design / HTML)

Drop Claude Design exports here so Cursor agents can find them:

- **Standalone HTML** from Export → save as `screen-name.html`
- **Screenshots** as `screen-name.png`
- **Notes** optional `screen-name.md` (layout deltas vs production)

The agent rule `.cursor/rules/design-system.mdc` points here automatically when building UI.

Primary spec remains [`../README.md`](../README.md) (tokens, components, IA).

**Interactive prototype (authoritative layouts):** [`../design_handoff_greenerherd_app/README.md`](../design_handoff_greenerherd_app/README.md) — open `GreenerHerd App.html` or read `src/screens/*.jsx` when implementing Flutter UI.
